X Instructions

  1. Start making notes by typing in the green textbox
  2. Click/tap to make text underlined (click again to disable)
  3. Click/tap to make text bold, (click again to disable)
  4. Click/tap Copy to copy all text in the textbox to clipboard.
  5. Click/tap Save to save a note to Saved Notes
  6. Click/tap Edit to edit the saved note.
  7. Click/tapX to close instructions.
